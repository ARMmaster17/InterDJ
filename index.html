What is InterDJ?

InterDJ is a service you can install on your own server to host your own internet radio shows. It's as simple as copying the files, creating a settings file, and uploading music.

How is InterDJ licensed?

We use the MIT license. This means you can use the software however you want, just make sure you attribute it back to us.

What state is InterDJ in?

InterDJ is currently in early development. Use at your own risk. To recieve updates, follow our GitHub page (Link at top of page).
